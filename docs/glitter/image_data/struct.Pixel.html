pub struct Pixel {
    pub r: u8,
    pub g: u8,
    pub b: u8,
    pub a: u8,
}

A single OpenGL color value, with u8 components laid out as a C struct in RGBA order. This type is the simplest implementation of Image2d, which allows it to be uploaded to a texture.

Fields

r: u8

The red component.

g: u8

The green component.

b: u8

The blue component.

a: u8

The alpha component.
